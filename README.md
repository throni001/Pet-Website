# 🐾 Paws & Play - Pet Care Website

A frontend-only pet care service website built with HTML and CSS. Designed for **desktop screens** — best viewed on laptops and monitors.

## Tech Stack

- HTML5
- CSS3
- Font Awesome Icons

## Preview

This is a static frontend website showcasing grooming, daycare, and boarding services for pets. It includes sections for:

- Services (grooming, daycare, boarding, etc.)
- About / Our Story
- Happy Pets Gallery
- Customer Reviews
- Call-to-action booking

## Note

> This site is optimized for **desktop viewing**. It may not render properly on mobile devices or small screens.
